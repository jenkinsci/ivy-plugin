The name of the property used to pass the names of the changed modules to the build.

Only required when "Incremental build" is selected and "Build modules as separate jobs" is not selected. If left empty in this case, then the default value of "hudson.ivy.changedModules" will be used.

The property specified here can then be used in your branch-level build script for the "leaf" attribute of the "ivy:buildlist" task.
